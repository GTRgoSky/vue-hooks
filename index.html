vue-hooks

基于composition API的可组合逻辑复用函数集

开始 →

Composable

利用Vue3 composition API 来构建逻辑复用函数，极大化了逻辑的组合能力

TypeScript支持

全部采用TypeScript实现，有良好的类型推导支持

# 安装和使用

  • @xuguo/vue-hooks实现基于Vue3 composition API的hooks函数集
// 安装依赖
npm install @xuguo/vue-hooks @xuguo/sku vue-demi
// 如果您使用的是vue2.x 您还需要安装
npm install @vue/composition-api

// 使用
import { useScroll } from '@xuguo/sku'
